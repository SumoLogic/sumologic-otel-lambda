# Sumo Logic Distribution for OpenTelemetry Python Lambda Layer v1.32.0

Sumo Logic lambda layers support:

- `Python` in versions `3.9, 3.10, 3.11, 3.12` and `3.13` runtimes
- `x86_64` and `arm64` architectures
