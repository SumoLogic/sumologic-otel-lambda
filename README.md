# Sumo Logic Distribution for OpenTelemetry Lambda Layers

`sumologic-otel-lambda` publishes preconfigured [OpenTelemetry Lambda](https://github.com/open-telemetry/opentelemetry-lambda) layers which provide instrumentation for AWS Lambda functions.
Released `sumologic-otel-lambda` layers are available:

- Java wrapper layer contains OpenTelemetry Java `v2.19.0` and OpenTelemetry Collector `v0.132.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-java-v2.19.0/java/README.md).

- NodeJS layer contains OpenTelemetry JavaScript SDK `v2.2.0` and OpenTelemetry Collector `v0.138.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-nodejs-v2.0.1/nodejs/README.md).

- Python layer contains OpenTelemetry Python SDK `v1.38.0` with instrumentation `v0.59b0` and OpenTelemetry Collector `v0.138.0`. Please see list of [lambda layers](https://github.com/SumoLogic/sumologic-otel-lambda/blob/release-python-v1.38.0/python/README.md).
